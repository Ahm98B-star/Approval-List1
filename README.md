## 🚀 Overview
This dashboard allows procurement managers to:
- Enter and manage **PO Approvals** and **Advance Approvals**.
- View automated SAR/Cur conversion rates.
- Dispatch professional, formatted email reports to managers via **EmailJS**.
- Collaborate in real-time using the **Supabase** backend.

## 🛠️ Project Structure
- `index.html`: Modern, responsive dashboard (Day/Dark mode supported).
- `style.css`: Premium Glassmorphism design system.
- `script.js`: Real-time data synchronization and Email engine.
- `.gitignore`: Prevents temporary and backup files from being uploaded.

## 📦 How to Deploy (Private)
1. Create a **PRIVATE** repository on GitHub.
2. Upload these files.
3. Invite team members as **Collaborators**.

## 🌓 Themes
The dashboard supports a **Professional Day Mode** and a **Sleek Dark Mode**. Toggle the theme using the button in the header.
